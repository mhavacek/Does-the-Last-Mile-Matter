# Data Instructions

Raw NHTS data are not redistributed in this repository.

1. Go to <https://nhts.ornl.gov>.
2. Download the 2017 NHTS Survey Data in CSV format.
3. Extract the files locally.
4. Place these files in `data/raw/2017/`:

```text
trippub.csv
perpub.csv
hhpub.csv
vehpub.csv
```

Optional files currently present in the local workspace, such as 2022 NHTS files or 2017 trip-chain files, are not required for the main analysis. The main analysis uses 2017 survey data because the 2017 trip file contains `TRACCTM`, `TREGRTM`, `TRACC_*`, and `TREGR_*` access/egress variables.

The processed file `data/processed/nhts_transit_last_mile_analysis.csv` is generated by:

```bash
.venv/bin/python src/02_prepare_analysis_dataset.py
```
